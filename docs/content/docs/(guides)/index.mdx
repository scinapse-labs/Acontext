---
title: "What is Acontext?"
description: "Skill memory for AI agents — learns from runs, writes Markdown skill files, reuses them on the next run"
---
Acontext is skill memory for AI agents. It automatically captures learnings from agent runs, stores them as Markdown files, and lets agents reuse those skills on the next run.
